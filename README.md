Bookhaven is a community book store website built using Angular for the frontend, Node.js for the backend, Socket.io for real-time chat functionality, and MongoDB as the database. It allows users to share posts about their books, interact with others, like and wishlist books, and reserve books with the owner's approval. The application also includes an admin login and dashboard with tables and charts for efficient management. Bookhaven aims to create a thriving community for book enthusiasts to connect, share, and explore a diverse collection of books.
